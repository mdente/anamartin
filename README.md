# anamartin.uy — GitHub Pages

Sitio estático simple en HTML, CSS y JavaScript.

## Archivos

- `index.html`: estructura, SEO, Open Graph y datos estructurados.
- `style.css`: visual negro, sombras, degradé animado y responsive.
- `script.js`: animación suave de interacción.
- `assets/ana-martin.webp`: imagen optimizada.
- `assets/ana-martin.jpg`: fallback y Open Graph.
- `robots.txt` y `sitemap.xml`: apoyo básico para indexación.
- `CNAME`: dominio personalizado para GitHub Pages.

## Publicación en GitHub Pages

1. Crear un repositorio, por ejemplo `anamartin-uy`.
2. Subir todos estos archivos en la raíz del repositorio.
3. Ir a `Settings > Pages`.
4. En `Build and deployment`, seleccionar `Deploy from a branch`.
5. Elegir rama `main` y carpeta `/root`.
6. Guardar.
7. Configurar el dominio `anamartin.uy` en DNS apuntando a GitHub Pages.
